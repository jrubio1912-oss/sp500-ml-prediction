# 📊 SP500 ML Prediction

## 📌 Descripción

Este proyecto tiene como objetivo predecir si el índice S&P 500 va a subir o bajar utilizando técnicas de Machine Learning.

## 🎯 Objetivo

Desarrollar un modelo de clasificación binaria que permita anticipar la dirección del mercado (sube o baja).

## 📂 Dataset

Se utiliza un dataset de datos históricos del S&P 500 en frecuencia horaria (H1), que incluye variables como precio Bid/Ask y volumen.

## 🧠 Enfoque del modelo

Se abordará el problema como una tarea de clasificación supervisada, utilizando modelos como:

* Logistic Regression
* Random Forest

## ⚙️ Tecnologías

* Python
* Pandas
* Scikit-learn
* Matplotlib

## 📁 Estructura del repositorio

El proyecto se organiza en una estructura modular que permite separar datos, análisis y lógica del modelo:

* **data/**: contiene los datos del proyecto, separados en datos crudos (`raw`) y procesados (`processed`).
* **notebooks/**: incluye notebooks de análisis exploratorio y pruebas iniciales.
* **src/**: contiene el código fuente del sistema, incluyendo preprocesamiento, generación de features y modelos.
* **README.md**: documentación general del proyecto.
* **requirements.txt**: librerías necesarias para reproducir el entorno.

Esta organización facilita la escalabilidad, mantenimiento y reproducibilidad del proyecto.
